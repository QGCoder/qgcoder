gcoder - A G-code editing Qt5 GUI
=================================

## Installation

```bash
gh repo clone QGCoder/gcoder
qmake
make
./gcoder
```

## Overview

To be written, see proof of concept [Video](https://www.youtube.com/watch?v=9D3hMXP5-QM).
